# project\_smartcampus

Project Distributed  Application Development



**CLONE REPO INTO LAPTOP**

To clone using windows cmd, run this command 

1. navigate to folder to locate the repo - cd \[folder path]
2. get repo link and clone - git clone https://github.com/nadiahafiz/project\_smartcampus.git



To clone using GitHub desktop 

1. File > Clone repository 
2. Choose repo 
3. On local path, browse to where the repo will be located
4. Click clone 



**PROJECT IMPORT**

There's several way to import the project, most safe way are

1. Open eclipse, create new workspace
2. General > existing project into workspace
3. Browser project folder , click next 
4. project successfully imported 
5. Clean \& update project



**SETUP**

1. Java version - jdk 17
2. Springboot version - 3.2.5
3. Database - h2
